Searching Users:

  • All and any users returned by any search do not include yourself.
  • All and any users returned by any search allow you to look at most of their profile information by pressing the "..." more menu to the right of their name. Info includes service types and locations.
  • All search result users can be shown with markers on the same map when pressing "Show on map".

To search for users, you:

  • Need to find a user by name

Search by choosing "All users by name", and leaving the name blank. This will return all users in name order. Name entered can be partial or full (FirstNameLastName). and will return all users with that text in their name(s).

  • Need to find a user by their residence location

Choose one of the complete list of residence locations given by all users, that are not yourself. Note that some users may not have registered a residence type location for themselves.

  • Need to find a user by their non-residence location

Choose one of the complete list of non-residence locations given by all users, that are not yourself. Note that some users may not have registered a non-residence type location for themselves.

  • Need to find a user nearest either of my profile locations or my current location

Choose an option with "User by air distance". Note that the current physical location can take a minute longer, since your device has to determine exactly where you area first. All users will appear in results, but ordered by closest to furtherest by the closest profile location that they registered.

  • Need to find users close to a particular location

Search "All Users near location name" and enter a location that is specific enough to pass a review based on geocoding. In other words, if you don't get an expected result based on your entered location, maybe you need to add the state or region to that location. The system doesn't check for review status, as it does when entered on your profile.

  • Need to find users that are members of one chosen from a list of military service channels (not including yourself)

Search service channels of US Military type. The options are:

  • All existing military service channels
  • Each individual military service channel separately

Results returned are users that are members of those channel(s)

  • Need to find users that are members of one chosen from a list of first responder channels (not including yourself)

Search service channels of first responder type. The options are:

  • All existing first responder service channels
  • Each individual first responder service channel separately

Results returned are users that are members of those channel(s)

  • Need to find all users who don't have a service type on their profile

Search "Users with no service type in profile" Note that the user(s) returned may have joined one or more service channels, even though they don't technically identify as that service type.
